## 🚩 四校報到處工作人員
**主要任務：預先報名者報到**

* **操作流程：**
  1. 開啟 [報到系統網址]()
  2. 登錄模式：**簽到模式**
  3. 執行：掃描醫師名牌上的 **QR Code**。
  4. 確認：畫面顯示「報到成功」即可。
* **異常處理：**
  * 若 QR Code 無法讀取，請引導醫師改報「姓名及密碼」手動登錄。

## 📝 學術組工作人員
**主要任務：處理現場報名與牙材券核發**

* **任務 A：現場報名處理**
  1. 引導醫師掃描現場報名 QR Code。
  2. 醫師完成填寫後，核對其身分證件。
* **任務 B：核發牙材券**
  1. 開啟 [牙材券核發系統]()
  2. 輸入醫師姓名或編號，點選 **「核發基本額度」**。
  3. 告知醫師可至系統查詢 QR Code 截圖備用。

---

## 💰 財務組工作人員
**主要任務：牙材券加購登錄、廠商結算發款**

::: tip 權限要求
本組別需具備 **「行政授權碼」** 方可操作結算功能。
:::

* **任務 A：牙材券加購**
  1. 確認收到醫師現金後，開啟 [秘書登錄系統]()。
  2. 輸入金額完成加購，並同步發放實體或數位抽獎券。
* **任務 B：廠商現場結算**
  1. 開啟 [廠商結算系統]()。
  2. 選取要結算的廠商。
  3. 核對系統內「未結算金額」與廠商提供之數據是否相符。
  4. 確認無誤後，點選 **「執行發款結算」**。

---

## 🎁 場地組工作人員
**主要任務：中獎牙材券登錄**

* **操作流程：**
  1. 確認中獎醫師身分與獎項。
  2. 開啟 [中獎登錄系統]()。
  3. 掃描醫師 QR Code 或輸入姓名。
  4. 登錄中獎額度，並引導醫師確認「醫師查詢看板」餘額是否入帳。

---

## 🏬 廠商工作人員
**主要任務：查詢即時業績、確認結算明細**

* **操作說明：**
  1. 請使用專屬之 [廠商查詢網址]() (需帶金鑰參數)。
  2. 進入看板後可查看：
     * **總業績**：今日累積所有訂單。
     * **已結算**：大會已撥款部分。
     * **未結算**：待向財務組領取之款項。
  3. 如需核對明細，請往下滑動查看簡化後之交易時間。

---

## 👨‍⚕️ 醫師身分說明
**對象：與會醫師**

### 1. 預先報名者
* 憑大會發放之 QR Code 名牌即可完成報到與後續所有消費。

### 2. 現場報名者
* 完成現場報名後，請至 [醫師查詢系統]() 登入。
* 建議將查詢到的 **專屬 QR Code 截圖** 存入手機相簿，消費時出示即可。
* 若手機沒電，可向工作人員報「姓名+密碼」進行消費。
