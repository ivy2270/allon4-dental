## 🚩 四校報到處工作人員 {#registration}
**主要任務：預先報名者報到**

* **操作流程：**
  1. 開啟 [報到系統網址](https://app.allon4-dental.tw/2026/gate.html?role=checkin)
  2. 登錄模式：**簽到模式**
  3. 執行：掃描醫師名牌上的 **QR Code**。
  4. 確認：畫面顯示「報到成功」即可。

---

## 📝 學術組工作人員 {#academic}
**主要任務：處理現場報名與牙材券核發**

* **任務 A：現場報名處理**
  1. 引導醫師掃描現場報名 QR Code。
  2. 醫師完成填寫時會自動核對是否已預先報名，如果身份證字號已存在，請引導醫師直接用姓名+密碼登入查詢。
* **任務 B：核發牙材券**
  1. 現場報名完成後，開啟 [牙材券核發系統](https://app.allon4-dental.tw/2026/signup-confirm.html)
  2. 選擇要核發牙材券的醫師姓名，依報名身份輸入核發金額，點選 **「核發基本額度」**。
  3. 告知醫師可至系統查詢 QR Code 截圖備用。

---

## 💰 財務組工作人員 {#finance}
**主要任務：牙材券加購登錄、廠商結算發款**

::: tip 權限要求
本組別需具備 **「行政授權碼」** 方可操作結算功能。
:::

* **任務 A：牙材券加購**
  1. 確認收到醫師現金後，開啟 [牙材券加購系統](https://app.allon4-dental.tw/2026/add-on.html)。
  2. 輸入金額完成加購，並同步發放數位抽獎券，醫師可在個人查詢中看到抽獎券編號。
* **任務 B：廠商現場結算**
  1. 開啟 [廠商結算系統](https://app.allon4-dental.tw/2026/secretary.html)。
  2. 選取要結算的廠商。
  3. 核對系統內「未結算金額」與廠商提供之數據是否相符。
  4. 確認無誤，點清現金或支票後，點選 **「執行發款結算」**。

## 🎁 場地組工作人員 {#venue}
**主要任務：中獎牙材券登錄**

* **操作流程：**
  1. 確認中獎醫師身分與獎項。
  2. 開啟 [中獎登錄系統](https://app.allon4-dental.tw/2026/add-on.html)。
  3. 輸入醫師姓名搜尋+登錄中獎金額，並引導醫師確認「醫師查詢看板」中獎金額是否入帳。

---

## 🏬 廠商工作人員 {#vendor}
**主要任務：醫師消費、查詢即時業績、確認結算明細**

* **操作說明：**
  1. 請先用專屬 [廠商消費網址](https://app.allon4-dental.tw/2026/gate.html?vendor=03旭達) 登錄身份。
  2. 掃描醫師 QR Code 進入消費表單，會自動代入醫師姓名、報名序號、廠商名稱，輸入金額及**醫師密碼（身份證後四碼）**即可完成消費。
  3. 進入[廠商專屬看板](https://app.allon4-dental.tw/2026/vendor_query.html?v=03%E6%97%AD%E9%81%94&key=22C38F49)後可查看：
     * **總業績**：今日累積所有訂單。
     * **已結算**：大會已撥款部分。
     * **未結算**：待向財務組領取之款項。
  4. 如需核對明細，請往下滑動查看。

## 👨‍⚕️ 醫師身分說明 {#doctor}
**對象：與會醫師**

### 1. 預先報名者
* 憑大會發放之 QR Code 名牌即可完成報到與後續所有消費。
* 直接掃描 QR Code 即可登入後台查詢牙材券餘額、報到狀態、消費記錄、抽獎券號碼等。

### 2. 現場報名者
* 完成現場報名後，請至 [醫師查詢系統](https://app.allon4-dental.tw/2026/gate.html) 登入。
* 建議將查詢到的 **專屬 QR Code 截圖** 存入手機相簿，消費時出示即可。
* 若手機沒電，可向工作人員報「姓名+密碼」進行消費。
